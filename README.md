# Nirvana — Indian Restaurant Website

A clean, premium, mobile-first brochure site for **Nirvana**, a Halal Indian restaurant in Houston.
Built with plain **HTML, CSS and JavaScript** — no build step, no framework, no dependencies.

## Run it

Just open `index.html` in a browser. For correct routing of the shared header/footer and to
avoid any `file://` quirks, serve it locally:

```bash
# from this folder
python -m http.server 8000
# then visit http://localhost:8000
```

## Structure

```
index.html        Home (flagship — links to everything)
about.html        Our story, mission/vision/values, team
menu.html         À la carte (sticky category nav) + daily buffet
catering.html     Landing page: services, gallery, stat counters, inquiry form
blogs.html        Article index (card grid)
offers.html       What's New / Offers (campaign-ready, stands alone)
gallery.html      Filterable masonry gallery + lightbox

css/styles.css    The full design system (navy + gold tokens)
js/config.js      ⭐ SINGLE SOURCE OF TRUTH — edit business details here
js/main.js        Shared header/footer/sticky-bar injection + all behaviors
assets/favicon.svg
robots.txt · sitemap.xml
```

## Editing content (no framework needed)

- **Ordering URL, phone, address, hours, social, buffet info** → `js/config.js`.
  Change `TOAST_ORDER_URL` in **one** place; every "Order Now" button uses it.
- **Menu items / offers / blog cards / gallery images** → edit the relevant `.html` directly.
  Each menu item is one `.menu-item` block; each offer/post is one `.card` / `.post-card`.
- The header, footer and mobile action bar are generated by `js/main.js` so the nav and
  business details stay consistent across all pages.

## What's implemented (per the brief)

- Navy + gold design system, editorial section bands, alternating dark/light surfaces.
- Cormorant Garamond (display) + Inter (body), fluid type scale.
- Transparent → solid-on-scroll header; full-screen mobile drawer; persistent **Order Now**.
- Mobile sticky action bar (Order / Call / Directions) that hides on scroll-down.
- **100% Halal** badge and **Buffet** promotion repeated site-wide.
- Toast integration: every Order Now opens `TOAST_ORDER_URL` in a new tab + fires an
  `order_now_click` analytics event (gtag / dataLayer ready).
- Minimal motion: a single quiet fade-and-rise on scroll, gentle hovers — all disabled under
  `prefers-reduced-motion`.
- Catering: services grid, gallery w/ lightbox, animated stat counters (reduced-motion safe),
  validated inquiry form with honeypot + designed success state.
- Gallery: filterable masonry, keyboard-accessible lightbox with arrow-key nav.
- Accessibility: skip link, focus-visible gold rings, ARIA on drawer/lightbox, ≥44px targets.
- SEO: per-page title/description/canonical/OG, semantic headings, JSON-LD
  (`Restaurant`, `Menu`, `AggregateRating`), `robots.txt`, `sitemap.xml`.
- Performance: preconnect + preloaded hero, lazy-loaded images with explicit dimensions,
  `font-display: swap`, tiny dependency-free JS.
